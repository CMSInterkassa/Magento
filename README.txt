Для установки в ручном режиме платежного модуля Interkassa необходимо произвести следующие действия:

1. Содержимое папки Interkassa скопировать в папку `{корень сайта}/app/`

2. Зайти в админку, перейти в раздел "Система" -> "Конфигурация" ( "System" -> "Configuration" )

3.Открыть вкладку "методы оплаты" ( "Payment methods" )

4.Выбрать модуль Interkassa, включить модуль и ввести все настройки
*Все настройки можно взять на сайте Интеркассы
(идентификатор магазина в списке ваших кас, тестовый и секретный ключ во вкладке безопасность)

5. В настройках вашей кассы на сайте Интеркассы ,в разделе Интерфейс:

    Заберите галочки "Разрешить переопределять в запросе" для всех полей

    URL успешной оплаты:http://yourMagentosite.com/checkout/onepage/success/
    URL неуспешной оплаты:http://yourMagentosite.com/checkout/onepage/failure/
    URL ожидания проведения платежа:http://yourMagentosite.com/checkout/onepage/failure/
    URL взаимодействия:http://yourMagentosite.com/Interkassa/response

6.В целях безопасности советуем во вкладке Безопасность включить проверку цифровой подписи.
