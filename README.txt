Для установки в ручном режиме платежного модуля Interkassa необходимо произвести следующие действия:

1.Убедитесь в соответствии версий модуля и вашей CMS, они должны совпадать.

2. Содержимое папки Interkassa2 скопировать в папку `{корень сайта}/app/`

3. Зайти в админку, перейти в раздел "Система" -> "Конфигурация" ( "System" -> "Configuration" )

4.Открыть вкладку "методы оплаты" ( "Payment methods" )

5.Выбрать модуль Interkassa, включить модуль и ввести все настройки
*Все настройки можно взять на сайте Интеркассы
(идентификатор магазина в списке ваших кас, тестовый и секретный ключ во вкладке безопасность)

6. В настройках вашей кассы на сайте Интеркассы ,в разделе Интерфейс:

    Заберите галочки "Разрешить переопределять в запросе" для всех полей

    URL успешной оплаты:http://yourMagentosite.com/checkout/onepage/success/
    URL неуспешной оплаты:http://yourMagentosite.com/checkout/onepage/failure/
    URL ожидания проведения платежа:http://yourMagentosite.com/checkout/onepage/failure/
    URL взаимодействия:http://yourMagentosite.com/Interkassa/response

7.В целях безопасности советуем во вкладке Безопасность включить проверку цифровой подписи.


!!!
Для тестирования платежей в кассе используйте тестовую валюту XTS (тестовая платежная система), для это необходимо включить в кассе этот пункт, и далее при перенаправлении на сайт "Интеркассы" выбрать ее в качестве оплаты, далее система сгенерирует тестовую страничку со всеми видами ответа от сервера, это поможет вам настроить все необходимые параметры и убедиться в работоспособности вашей системы.
